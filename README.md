# Siddiq Food Hub Showcase

## Overview
Siddiq Food Hub is a university project demonstrating full-stack web development skills for a food ordering and delivery platform. This showcase highlights my ability to design and implement modern web applications with user-friendly interfaces and robust functionality.

## Purpose
This project was developed as part of my university coursework to demonstrate practical software development skills including:
- Front-end and back-end development
- Database design and management
- User interface/experience design
- Problem-solving and project planning

## Technologies Used
- Modern web technologies and frameworks
- Database management systems
- Version control with Git

## Features
- User authentication and authorization
- Menu browsing and ordering system
- Order management and tracking
- Responsive design for mobile and desktop

Customers can browse through vendor storefronts to explore menus, view food items and add food items to cart. 

### Cart & Checkout

![Cart Checkout](Customer/customer%20cart%20checkout.png)

The cart and checkout system allows customers to review their selected items, adjust quantities and complete their purchase.

### Customer Checkout Process

![Checkout Process 1](Customer/Customer%20Checkout%20process%201.png)

![Checkout Process 2](Customer/Customer%20Checkout%20process%202.png)

The checkout process guides customers through order confirmation, delivery details and payment.

### Order Confirmation

![Order Successful 1](Customer/Customer%20order%20place%20successful.png)

![Order Successful 2](Customer/Customer%20order%20place%20successful%202.png)

Upon successful order placement, customers receive a confirmation screen with their order details and tracking information.

### Order Tracking

![Order Tracking](Customer/cust%20complete%20order.png)

The order tracking feature displays the current status of customer orders. Vendors manually update the order status which is reflected in real-time on the customer's order tracking page.

### Loyalty Program

![Loyalty Program 1](Customer/Customer%20loyalty%20program%201.png)

![Loyalty Program 2](Customer/Customer%20loyalty%20program%202.png)

The loyalty program rewards customers for their continued patronage. Customers earn points with each purchase and can redeem free meal voucher. 

### Free Meal Voucher Redemption

![Free Meal Voucher Checkout](Customer/Free%20meal%20voucher%20checkout.png)

Customers can apply their earned free meal vouchers during the checkout process.

### Customer Feedback System

![Customer Feedback System](Customer/customer%20feedback%20system.png)

The feedback system allows customers to submit reviews, complaints or suggestions about their experience. All feedback is publicly displayed on the vendor's storefront page.

### Customer Rating System

![Customer Rating System](Customer/customer%20rating%20system.png)

Customers can rate individual food items from their completed orders. These ratings are displayed on the respective menu items on the vendor's storefront.
